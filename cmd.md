### Basic Guide of cmd

#### What is cmd?
cmd stands for comand, and is the comand console for windows, It allows to execute instructions avoinding the usage of graphic windows interfaces, usefull for browse folder, create files, execute programas, manage networks, among others.

#### How to execute cmd?

There are 3 ways:  
- Press Win + R, write cmd and push enter.  
- Write cmd in Windows Search  
- In a folder → Shift + rigth clic → “Open command window here
